	Version 1.1

	Created by Shiro_pb

	This mod adds a new spear to the game. 
	It has three charges that deplete as you hit foes. 
	To charge it back up just hit a centipede

    How to instal:
	
	This is a Partiality mod, so...
	follow this tutorial in Raindb.Net
	https://www.raindb.net/tutorials.html
	at the "Partiality Mods" section.

    known incompatibilities:

	-Lancer by Topicular
	-Beastmaster by Fyre
	
    Current version:

	-Charge is now preserved between cycles!		
	-Spears can be found in Scavenger's dens
	-No arena support
